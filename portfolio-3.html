HAPPIT

Description

Happit is a mobile app for iOS and Android that helps users to improve their mental, physical, and emotional well-being through goal setting and personal accountability. Features include a calendar and charts to easily visualize progress.

Technologies

This app is built with Ionic, ngCordova, and Angular. The server side makes use of Node.js, Express, Knex, and PostgreSQL. Additional tools used include Angular Charts, Angular Material Calendar, and Facebook OAuth.
